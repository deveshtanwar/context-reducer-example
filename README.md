# ToDo App

A simple ToDo application built with **React.js** using **Context API** and **useReducer** hook for state management.

## Features

- Add a new task
- Delete a task
- Manage tasks efficiently with a single global state

## Technologies Used

- **React.js** - JavaScript library for building user interfaces
- **Context API** - To provide and consume state globally
- **useReducer** - A hook for managing complex state logic
